1. pass the command line arguments to program: [r args]
2. set breakpoint with a condition: [break file:line if condition]
3. execute the next line: [s/n; (s: if there is a function calling, then enter the function; n: execute the function)]
4. enter a function: [s]
5. resume program after breakpoint: [c]
6. see value of variable: [p variable]
7. print value of a variable after each step: [display variable]
8. print a list of all variables and values in a function: [info variables]
9. exit: [q]
